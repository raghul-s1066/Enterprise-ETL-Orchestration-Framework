## 📌 Project Overview

This project implements an ETL (Extract, Transform, Load) pipeline using Apache Airflow to process a literacy rate dataset.

The pipeline:
- Extracts data from source
- Transforms and cleans the data
- Loads processed data
- Monitors workflow using Airflow
